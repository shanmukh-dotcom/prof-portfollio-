## ✨ Features

- **Dark Mode Aesthetic**: A deep, immersive dark theme with vibrant purple accents (`#6B40FF`) and subtle glowing ambient lights.
- **Glassmorphism UI**: Beautiful frosted-glass cards with backdrop blurs and glowing borders.
- **Dynamic Animations**: Custom CSS keyframes including a floating interactive avatar, smooth hover states, and continuous auto-scrolling galleries.
- **Fully Responsive**: Flawless layout across desktops, tablets, and mobile devices.
- **Integrated Contact System**: A "Send Message" form that seamlessly hooks into the native email client using JavaScript.

## 🛠️ Tech Stack

- **Frontend**: Pure HTML5, CSS3, and Vanilla JavaScript (Zero heavy frameworks, highly optimized).
- **Typography**: [Inter](https://fonts.google.com/specimen/Inter) by Google Fonts.
- **Icons**: 
  - [FontAwesome 6](https://fontawesome.com/) for UI elements and social icons.
  - [DevIcon](https://devicon.dev/) for precise tech-stack logos.

## 🚀 Running Locally

Because this project is built with Vanilla web technologies, running it locally takes less than a second.

1. Clone the repository:
   ```bash
   git clone https://github.com/shanmukh-dotcom/prof-portfollio-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd prof-portfollio-
   ```
3. Open `index.html` in your favorite web browser. That's it! 
   *(Optionally, use an extension like VS Code Live Server for hot-reloading).*
